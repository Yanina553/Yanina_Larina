# Игра "Змейка"

Классическая игра "Змейка", созданная в рамках лабораторной работы по парному программированию.

## Описание проекта
Это простая аркадная игра, в которой игрок управляет змейкой, ползающей по экрану. Змейка постоянно движется и увеличивается в длину, когда поедает еду. Игра заканчивается при столкновении со стеной или собственным хвостом.

## Как запустить игру

### Требования
- Python 3.x
- Библиотека Pygame

### Установка и запуск
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Yanina553/Yanina_Larina.git
2. Перейдите в папку с игрой:

bash
cd Yanina_Larina
3. Установите необходимые зависимости:

bash
pip install -r requirements.txt
4. Запустите игру:

bash
python snake_game.py
Управление
Стрелка влево (←) — повернуть налево
Стрелка вправо (→) — повернуть направо
Стрелка вверх (↑) — повернуть вверх
Стрелка вниз (↓) — повернуть вниз
C — начать заново после проигрыша
Q — выйти из игры

Правила игры
Змейка движется непрерывно
Съедайте красные квадраты, чтобы увеличить длину и получить очки
Не врезайтесь в стены и в собственный хвост
С каждым съеденным кусочком змейка становится длиннее, а игра сложнее
